Je souhaite installer l'application sur ma télévision

L'application officielle Jellyfin est disponible sur les box opérateurs suivantes :

  • Bbox via Play Store
  • À partir de la SFR Box 8 avec Connect TV Android

L'application officielle Jellyfin est disponible sur les Smart TV et systèmes suivants :

  • Télévisions sous Amazon FireTV et FireTV Stick
  • Télévisions LG après 2010
  • Télévisions Sony Bravia sous Google TV
  • Télévisions Hisense (installation manuelle)
  • Télévisions Samsung sous Tizen (installation manuelle)
  • Add-on JellyCon pour Kodi
  • Add-on Jellyfin pour Kodi

Pour les opérateurs et ou les autres systèmes ne disposant pas d'application dédiée :
